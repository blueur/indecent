# indecent

A rolling dice application.

## Run

```shell
# Web
flutter run -d chrome
```

## Build

### Flutter

```shell
flutter run --release
# Android
flutter build appbundle
# Web
flutter build web
```

### Miscellaneous

```shell
# generate icons
flutter pub run flutter_launcher_icons:main
```

## Deploy

```shell
# Firebase
firebase deploy
```
